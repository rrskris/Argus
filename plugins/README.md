# plugins/ — reserved, not functional

These directories hold descriptor manifests (`plugin.yaml`) sketching a future
integration-plugin system. There is no plugin runtime today — nothing here is
loaded, executed, or shipped. Integrations that actually exist are documented in
[docs/ci-integration.md](../docs/ci-integration.md) and
[docs/trivy-grype-integration.md](../docs/trivy-grype-integration.md) (design).
